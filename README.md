# CIVICeye

CIVICeye is a modern web application that enables citizens to report, track, and ensure resolution of public issues using evidence-based accountability. Unlike traditional complaint systems, CivicProof ensures that: Every issue is backed by photo/video proof Complaints cannot be marked resolved without verified evidence or citizen confirmation.

🌟 Features 📸 Proof-Based Reporting Upload images/videos along with issue descriptions and location. 🔐 Authentication System Secure login/signup for users and admins. 📊 User Dashboard Track all submitted complaints and their statuses. 📝 Complaint Feed Public feed to view reported issues in real-time. ⏳ Tracking Timeline Monitor progress from submission → verification → resolution. 🛠️ Admin Panel Authorities can: Review complaints Upload resolution proof Update status 📍 Geolocation Support Attach precise location data to each complaint.
